# Chapter 12: Deep Learning for Time Series Forecasting

## Summary

This chapter explores the application of deep learning architectures to time series, including LSTMs, TCNs, and Transformers (N-HiTS), using frameworks like Keras, Darts, and NeuralForecast.

## Recipes

- Understanding artificial neural networks
- Forecasting with LSTM (Keras, Darts, and Neural Forecast)
- Handling multivariate time series and exogenous variables
- Hyperparameter tuning for deep learning forecasters
- Forecasting with TCNs
- Forecasting with Transformers (N-HITS)
